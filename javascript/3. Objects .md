# Object

object는 배열과 비슷한 역할을 하지만 하나 하나에 의미를 부여할 떄 유용하게 쓸 수 있다.

object의 리터럴 방식은 **{ }** 중괄호를 사용한다.
``` javascript
const player = {
  name: "nico",
  points: 10,
  fat: true,
};
console.log(player);
```
이때 Object의 값을 바꾸는 방법은
``` javascript
player.fat = false;
player.points = player.points+15;
console.log(player);
```
또 Object에 값을 추가하는 방법은 
``` javascript
player.lastName = "potato";
```
이런식으로 작성하여 추가할 수 있다.
